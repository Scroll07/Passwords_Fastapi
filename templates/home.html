Password management for the command line.

Pas CLI is a local-first password manager for developers. Store credentials in an encrypted vault on your machine, with optional API sync for cross-machine backup.

Get started Read about the product 
git clone https://github.com/Scroll07/password_manager
cd password_manager && ./realise.sh

pas copy github
Password copied to clipboard.

pas api upload
Vault backed up to API.

What Pas CLI does

Pas CLI is split into two parts. The CLI client handles all local vault operations — adding, searching, copying, and editing credentials — entirely on your machine.

The FastAPI backend is optional and adds registration, login, and encrypted vault backup sync so you can restore your vault on another machine.

The vault is encrypted and protected by a master password that never leaves your device.

Common commands

pas add github -u user --gen Add an entry and auto-generate a password
pas copy github Copy a password to clipboard
pas find gmail.com Search entries by pattern
pas export backup.json Export vault to a local JSON file
pas api upload Back up vault to the API
pas api download Restore vault from the API on a new machine
