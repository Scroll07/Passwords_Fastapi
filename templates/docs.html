Pas CLI Reference

Complete reference for installation, commands, and the backend API.

Installation

Clone the repository and run the setup script. Python 3.8+ is required.

git clone https://github.com/Scroll07/password_manager.git
cd password_manager
./realise.sh

Quick start

Register on the API (optional), then add your first entry:

pas api register
pas add test -u test -p test --note test

Base commands

All local vault operations. These work without an API account.

Command Description Example
pas add Add entry pas add github -u user --gen
pas list List all entries pas list
pas get Show entry detail pas get github
pas copy Copy password to clipboard pas copy 2
pas edit Edit entry pas edit 1 -p newpass
pas del Delete entry pas del github
pas find Search entries pas find @gmail
pas reset-session Reset session token pas reset-session
pas export Export vault to JSON pas export backup.json
pas import Import vault from JSON pas import backup.json
pas change-master Change master password pas change-master
pas create-key Generate a password pas create-key -l 32
pas config Configure API token pas config --token

API commands

Commands that interact with the FastAPI backend. Require a registered account.

Command Description
pas api register Register a new account on the API
pas api login Log in and receive JWT tokens
pas api upload Upload vault backup to the API
pas api download Download vault backup from the API
pas api delete Delete a vault backup

Backend endpoints

REST endpoints exposed by the FastAPI backend. All /backups routes require a valid JWT access token in the Authorization header.

Method Endpoint Description
POST /register Register a new user
POST /login Login and receive JWT tokens
GET /refresh Refresh the access token
GET /backups List all backups for the user
POST /backups/upload Upload an encrypted vault backup
POST /backups/download Download a vault backup
DELETE /backups/{id} Delete a specific backup
